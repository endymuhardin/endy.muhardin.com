Pengampu program pelatihan Junior Site Reliability Engineering untuk PT ArtaJasa Pembayaran Elektronis. Program berjalan dalam beberapa batch: Batch 1 (September 2025), Batch 2 (November 2025), dan batch 2026 (April 2026).

Kurikulum 5 hari mencakup:

- **SRE Fundamentals & Service Level** — SRE vs IT Operations, SLA/SLO/SLI, error budget.
- **CLI, Debugging & Networking** — Linux command-line, network diagnostics, troubleshooting latency transaksi.
- **VM, Docker & Alert Triaging** — konsep VM dan container, Docker fundamentals, image security, incident response.
- **Scripting & Hardening** — Bash dan Python untuk log parsing dan otomasi, security hardening, supply chain security (case study Java, Node.js, Go).
- **Redundancy, Testing & Incident Learning** — high availability, failover testing, post-incident review dan runbook.

Materi batch terbaru: [training-sre-junior-2026-01](https://github.com/endymuhardin/training-sre-junior-2026-01).
