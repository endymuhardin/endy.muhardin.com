Penelitian ini membahas otomasi pemrosesan data keuangan untuk usaha kecil menggunakan OCR. Solusi serverless mengintegrasikan platform Telegram dengan Google Cloud Vision API untuk mengekstrak data dari faktur dan struk transaksi, mencapai akurasi pengenalan teks di atas 90% pada kondisi pencahayaan normal.

Kesimpulannya: tool gratis dan infrastruktur berbasis cloud dapat menurunkan hambatan adopsi teknologi akuntansi bagi pelaku usaha mikro, mengurangi kesalahan input manual dan beban administratif.
